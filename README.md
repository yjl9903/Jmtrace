# Jmtrace

Trace Java memory access.

## Installation

## Usage

```text
$ jmtrace --help
Usage: jmtrace options_list
Arguments: 
    args -> Args for running jar (optional) { String }
Options: 
    --jar, -jar -> Input jar package (always required) { String }
    --verbose, -v [false] -> Enable verbose Log 
    --help, -h -> Usage info
```
